API Reference

REST calls you probably care about.

Establishes a session. Send both a username and a password. Returns a session_id that identifies this session. 
POST /authentication
Make a handshake with the server. Send a username and a target_system_id. Returns all the available data that Simplixity has. 
POST /handshake
Sends an info response to Simplixity to authorize it to send the selected data to the target machine. 
POST /response

Users and Policies

Get all the users 
GET /user
Sample user JSON: 
{
    _id: {
        $oid: "541dfcdaee6da110ec9326f9"
    },
    conditions: [{ code: "123456789",
                   type: "allergy"}],
    organ_donor: true,
    policies: [{ $oid: "541dfcdaee6da110ec9326f8"}],
    employer: "Virginia Commonwealth University"
}
Get an individual user 
GET /user/[user_id]
Get an individual's personal data 
GET /person/[user_id]
Get an individual policy 
GET /policy[policy_id]
